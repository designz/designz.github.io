jQuery deferred对象

在高于1.5.0的jQuery版本$.ajax()返回的是

javascript
done(function(){}) //成功
fail(function(){}) //失败
reject() //改变deferred状态为失败
resolve() //改变deferred状态为成功

javascript
$.when(deferredObj1,defrredObj2).done(…).fail()

javascript
var a =function(){
var dtd = $.Deferred();
//….other code….//
return dtd.promise() //这里用promise()，是为了让返回的deferred对象屏蔽其改变状态的方法。即不能使用reject和resolve。
}

$.when().then(fna,fnb);````的写法，fna为完成状态的回调，fnb为未完成时的回调。
